## 🧠 What is Rabisco?

**Rabisco** is a creative tool designed to help artists overcome one of the most common blocks in drawing:

*"I want to practice… but I don’t know what to draw."*

The idea came from a real-life moment during a **drawing and painting course**, where several students shared the same struggle: having the discipline to practice, but lacking ideas.

So Rabisco was created as a simple and accessible way to generate inspiration and make practice easier 🎨


## ✨ Features

### 🎲 Random Drawing Prompts

- Generate ideas for drawings instantly
- Helps break creative blocks
- Encourages daily practice

### 🧠 Idea Exploration

- Mix different concepts and themes
- Stimulates creativity instead of repetition
- Great for beginners and experienced artists

### ⚡ Fast & Frictionless

- No login required
- No setup needed
- Open and start drawing

### 🎨 Minimal UI

- Clean interface focused on creativity
- No distractions — just ideas

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- Ruby 3+
- PostgreSQL

### Installation

1. Clone the repository

```
git clone https://github.com/devanaclimgo/rabisco.git
cd rabisco
```

2. Backend setup

```
cd backend
bundle install
rails db:create db:migrate
rails server
```

3. Frontend setup

```
cd frontend
npm install
npm run dev
```

4. Open your browser

```
http://localhost:5173
```
